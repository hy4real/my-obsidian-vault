# 【Arch零门槛】绝美Linux桌面，一键安装Niri+DMS

> [!info] 视频信息
> 林长枊Shorin709 / 3:54 / 2026-02-19 / [Bilibili 原链接](https://www.bilibili.com/video/BV1CHZaBrEF6/)

## 核心观点

这是一套面向 Arch Linux 的**开箱即用桌面配置**，以 Niri 为窗口管理器、DMS（基于 QuickShell）为桌面套件，作者通过杂交自己的 ShoryuNiri 配置与官方 DMS，补全了默认快捷键、中文输入法、主题配置等缺失，实现一条命令即可装好的绚丽 Linux 桌面。

## 核心组件

| 组件 | 说明 |
|------|------|
| **Niri** | 窗口管理器，优秀的布局逻辑，极低占用 + 流畅动画 |
| **DMS** | 基于 QuickShell 的桌面套件，提供现代桌面体验（任务栏、概览、壁纸、主题等） |
| **ShoryuNiri** | 作者自己的配置层，弥补 DMS 默认配置的不足 |

## DMS 的优缺点

**优点：**
- 目前完成度最高的 QuickShell 配置
- 图形化设置中心（显示器、壁纸、配色等），无需手改配置文件
- 精简，占用低于其他主流 QuickShell 配置

**缺点：**
- 默认快捷键难用
- 预装软件少，开箱即用前需要额外配置

## ShoryuNiri 补全的内容

- 中文输入法
- 更合理的快捷键设计
- 便利脚本（`mirror-update`、`sysup`、`clean` 等）
- 主题配置补全
- 常见 bug 修复
- 软件安装别名（`安装 <包名>` / `卸载 <包名>`）

## 安装步骤

> **前提**：一个刚安装好的 Arch Linux 或任意 Arch 衍生发行版

1. 在终端运行安装脚本（视频中展示，具体见作者文档）
2. 安装选项中选择 **Niri**（第二项）或 **ShoryuNiri + DMS**（第三项）
3. DMS 官方脚本阶段：必须选 **Niri** 和 **Kitty**
4. 安装结束后，按需在菜单中选择常用软件（Tab 选中，回车确认，N 跳过）

> [!tip] 网络问题
> 如果安装失败，查看作者文档中的网络解决办法

## 首次进入桌面的配置

1. `Super+E` 打开文件管理器，读取家目录的说明文档
2. 开始菜单 → DMS 设置 → 选择壁纸（作者预置了几张）
3. 页面底部开启**副本功能**（概览界面的模糊背景）
4. `Super+G` 或 `Super+O` 切换概览界面
5. 配色界面选择**自动**，跟随壁纸自动切换主题色
6. 页面底部**应用 GTK 主题**
7. 重启或注销后完成

## 常用命令速查

```bash
# 软件管理（pacman 别名）
安装 <包名>     # 等同 pacman -S
卸载 <包名>     # 等同 pacman -R

# 系统维护
mirror-update   # 更新镜像源
sysup           # 更新系统
clean           # 清理缓存
```
