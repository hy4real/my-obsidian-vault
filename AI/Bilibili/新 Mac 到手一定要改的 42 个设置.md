# 新 Mac 到手一定要改的 42 个设置

> [!info] 视频信息
> Raycal_Tech / 15:44 / 2026-05-06 / [链接](https://www.bilibili.com/video/BV1r2RWB6EQN)

## 核心观点

Mac 默认设置有很多不好用的地方，作者用 Mac 6 年总结了 42 个必改设置，覆盖触控板、Dock、菜单栏、Finder、效率提升和体验优化六个方面。核心思路是：改完这些设置后，Mac 从第一天就好用，甚至不需要鼠标。

## 1. 触控板设置

- **三指拖移**：辅助功能 → 指针控制 → 触控板 → 使用触控板进行拖移 → 三指拖移。解决默认拖移"移到头了还没到地方"的问题
- **轻点即放**：触控板设置打开"轻点一点按"，三指拖移到位后轻点一下立刻放下，不用等 1 秒吸附
- **灵敏度**：调到倒数第二格，默认速度太慢
- **点按强度**：调成弱，省力且安静
- **四指下滑显示窗口**：EPIC pose 打开后，四指下滑显示当前 app 所有窗口

## 2. Dock 栏设置

- 移除不常用软件，从启动台拖入常用软件
- **关闭最近使用**：桌面与程序坞 → 关闭"程序坞中显示建议的和最近使用的 app"
- **最小化到图标**：打开"将窗口最小化到应用程序图标"，窗口缩回各自 app 图标
- **下载文件夹**：改为网格显示、按添加日期排列，新下载永远排第一
- **Dock 隐藏**：Command+Option+D 快速开关隐藏

## 3. 菜单栏设置

- 删除不用的图标（搜索用 Command+空格，Siri 基本没用）
- **按住 Command 拖动**可删除/重排菜单栏图标
- **添加天气**到菜单栏，室内也能实时看天气
- **电池百分比**：系统设置 → 菜单栏 → 电池 → 显示百分比
- **时钟显示秒**：菜单栏 → 时钟 → 在时间中显示秒钟

## 4. Finder 设置

- **切换列表视图**：默认图标视图不方便翻找文件
- **显示硬盘/外接磁盘**：通用设置 → 勾选硬盘、外接磁盘、已连接服务器
- **新窗口打开常用文件夹**：通用设置 → 开启新访达窗口时改为自己的项目文件夹
- **边栏自定义**：勾选需要的收藏和位置
- **显示文件扩展名**：高级 → 显示所有文件扩展名
- **文件夹置顶**：高级 → 勾选"将文件夹保持在顶部"（桌面和 Finder 都勾）
- **自定义工具栏**：添加隔空投送和新建文件夹按钮
- **路径栏**：Command+Option+P 显示，右键+Option 可拷贝路径
- **状态栏**：Command+/ 显示文件个数和剩余空间

## 5. 提升效率设置

- **截图快捷键**：键盘快捷键 → 截屏 → 把"拷贝到剪贴板"改为 Command+Shift+A（单手可按）
- **输入法切换**：打开"用大写锁定键切换美国输入法"，快速中英切换
- **自动切换输入法**：根据软件自动切换，不用手动切
- **滚动手势缩放**：辅助功能 → 缩放 → 配合修饰键使用滚动手势，Ctrl+双指上下滑放大缩小
- **多窗口合并标签**：桌面与程序坞 → 打开文稿时首选标签页 → 始终
- **调度中心分组**：打开"使窗口按应用程序成组"，切换 app 一目了然
- **触发角**：右下角关屏幕、左下角屏保、右上角显示桌面。建议加修饰键防误触

## 6. 提升体验设置

- **关闭 iCloud 桌面同步**：iCloud → 关闭"同步桌面与文稿文件夹"，避免无意义占用空间
- **Apple Watch 解锁**：触控 ID 与密码 → 用 Apple Watch 解锁 Mac
- **更高分辨率**：显示器 → 更多空间，4K 显示器选 5K 渲染模式
- **充电上限 80%**：电池 → 充电上限设 80%（需 macOS 26.4+），保护电池寿命
- **AirPlay 接收器**：通用 → 隔空投送 → 打开隔空播放接收器，手机可 AirPlay 到电脑
- **滚动条仅滚动时显示**：外观 → 显示滚动条 → 滚动时
- **关闭点按桌面显示**：桌面与程序坞 → 点按墙纸显示桌面 → 改为"仅在台前调度中"
- **声音输出设备**：声音 → 播放声音效果设备改为所选输出设备，避免戴耳机时提示音外放
- **关闭开机声音**：声音设置 → 关闭启动时播放声音

> 使窗口按应用程序成组这个选项打开 这样他就会按照不同的app进行分类了 而且下面还会显示每一个app的图标 切换起来一目了然 可以做到非常高效的切换 软件 在桌面与程序录的设置的最下面 还有一个叫触发角的设置 在这里你可以设置 当鼠标移到屏幕四个角落的时候 所触发的操作 比如我会把右下角设置成关闭屏幕 左下角设置成屏保 鼠标只要移到右下角就可以关闭屏幕 左下角就可以进屏表 非常的方便 但是呢这样还有一个不足之处 就是非常容易误触 所以你可以给它在设置的时候加一个修饰键 比如说按住shift的时候再点这个 这样的话你就需要按住shift的同时
>
> 然后把鼠标移到右下角才能够触发这个操作了 然后右上角的设置我会设置成显示桌面 这样当我要拖动桌面上的东西的时候 可以先把鼠标碰一下右上角 然后在桌面拖住你要的文件 然后再碰一下右上角 就可以把桌面的东西 快速拖到你想用的app里面了 最后呢还有几个可以提升体验的设置 也一定要设置一下 来到设置
>
> 点击你的头像进入ICLOUD点查看 全部在ICLOUD云盘里面
>
> 把同步桌面与文稿文件夹给它关了 因为我桌面上放了很多我的视频项目 文件同步到ICLOUD就没有意义 大家如果默认只有5GB的空间的话 肯定也不想让桌面上乱七八糟的东西 占用ICLOUD空间吧 要是有用要同步的资料 你可以自己放到ICLOUD里面去 然后是在触控id与密码里面 如果你有apple watch 可以开启用apple watch来解锁Mac 如果说你用的是Mac mini或者新款的MACBOOKNEO 它没有touch id的话 你这样子解锁就会方便很多 然后在显示器的设置里面 我每次都会选择更多空间 这样的话更高的分辨率呢 就能显示更多的东西了 我觉得用着会更加舒服一点 我外接4K显示器的时候 我也会选择这个渲染5K分辨率的模式 然后如果你的MACBOOK经常是插电用的话 为了更好的延长你电池的寿命 在更新26.4版本之后 在电池设置充电里面 就可以设置充电上限到80%了 避免电池长期处在100%的电量 可以更好地保护电池
>
> 在通用隔空投送与连续互通里面 还可以打开隔空播放接收器 这样你用手机放歌的时候 就可以选择airplay到电脑播放了 比如说我的Mac mini连接了一对音响 我只要在放歌的时候选择用Mac mini来播放 就可以用音响来放歌了 有线音响直接就变无线音响了 有时候系统会卡bug
>
> 导致所有页面的滚动条都没办法自动消失 超级难受 遇到这样的情况 你可以在外观设置里面 把显示滚动条调成滚动时 这样的话只有你每次在滑动页面的时候 才会显示右边的滚动条 现在新的系统每次不小心点一下桌面 它就会显示桌面真的非常的烦人啊 我每次选中文件 想点一下空白的地方 它就显示桌面 看完下载文件夹 点一下空白的地方啊 就显示桌面 用完控制中心想关的时候 屏幕上点一下 哎它又显示桌面了 真的是受不了啦 如果你也觉得难受的话 一定要在桌面与程序五里面 把点按墙纸显示桌面 改成仅在台前调度中就好了 真不知道为什么默认要开启这么个选项啊 最后呢 如果你经常在图书馆教室之类的地方用电脑
>
