# Claude Style Obsidian Theme

一个受 Claude 视觉语言启发的 Obsidian 主题，提供浅色与深色模式、克制的陶土橙强调色、衬线正文阅读体验，以及适配 Style Settings 插件的主题变量。

## 安装

1. 下载或克隆本仓库。
2. 将仓库目录放到 Obsidian vault 的 `.obsidian/themes/Claude` 下。
3. 在 Obsidian 中打开 `设置 -> 外观 -> 主题`，选择 `Claude`。

## 文件

- `manifest.json`：Obsidian 主题元信息。
- `theme.css`：主题样式与 Style Settings 配置。

## 可调设置

安装 Style Settings 插件后，可以调整阅读宽度、正文行高、段落间距、界面密度、侧边栏浮动效果，以及编辑与阅读正文的字体风格。
