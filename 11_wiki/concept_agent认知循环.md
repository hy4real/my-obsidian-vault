# Agent 认知循环

> 感知 → 决策 → 行动 → 学习，四步循环构成 Agent 的运作内核。

## 四步循环

### 感知（Perception）
- 接收任务描述、上下文、反馈
- 理解问题，抽取关键信息
- 输出内部状态表示（task model）

### 决策（Decision）
- 评估可选方案
- 推理选择最优策略
- 生成具体行动序列

### 行动（Action）
- 调用工具，生成内容
- 监控执行状态
- 面对异常时纠正

### 学习（Learning）
- 反思评估行动结果
- 提取可复用经验
- 改进决策和规划策略

## 循环关系

```
感知 → 决策 → 行动 → 学习
  ↑                    ↓
  └────────────────────┘
```

学习的反馈改进感知和决策，形成持续提升的闭环。

## 应用示例

在 LLM-Wiki 的 Ingest 操作中：
- **感知** — 读取原始文档
- **决策** — 决定如何分类和链接
- **行动** — 创建 concept_*/guide_* 页面
- **学习** — 记录分类规律改进下次效率

## 与信息四层的区别

本页描述的是 Agent 的**运作过程**（怎么思考和行动）。[[concept_ai系统设计原则]] 中的四层机制描述的是**信息存储分层**（提示词/上下文/外部记忆/微调），是两个正交的维度。
