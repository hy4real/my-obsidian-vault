# 算法刷题计划 - Java + Python

## 目标
为27年春招做算法准备（LeetCode Medium/Hard，掌握常见考点）

## 为什么重要
- **27春招** = 算法面试必考
- **系统设计** = 需要扎实的数据结构和算法基础
- **竞争力** = 大厂都卡在算法题

## 学习阶段

### Phase 1: 基础强化（4月）
**目标**：掌握常见数据结构和经典算法模板

**关键主题**（Medium难度）：
- [ ] 数组 & 字符串（双指针、前缀和）
- [ ] 链表（反转、合并、快慢指针）
- [ ] 栈 & 队列（单调栈、滑动窗口）
- [ ] 树（DFS、BFS、二叉树性质）
- [ ] 图（DFS、BFS、拓扑排序）
- [ ] 动态规划（基础DP问题）

**题量**：15-20道/周，选择经典题

### Phase 2: 深度掌握（5月）
**目标**：掌握Hard难度题目，理解思维方式

**关键主题**（Hard难度）：
- [ ] 动态规划进阶（背包、状态转移）
- [ ] 图论进阶（最短路、最小生成树）
- [ ] 贪心算法
- [ ] 分治 & 回溯

**题量**：10-15道/周，重点理解思路

### Phase 3: 模拟面试（6月）
**目标**：在有时间限制下快速解题

**方式**：
- [ ] LeetCode Contests（周赛、双周赛）
- [ ] 模拟面试（1.5小时4道题）
- [ ] 讲述讲解（能讲清楚思路）

**题量**：5-10道/周，模拟真实面试

## 编程语言选择

### Java版
**优先级**：⭐⭐⭐⭐⭐
- 面试中文大厂常用Java
- 项目和算法同时提升（工程化实现）

**学习方式**：
- 用Java实现，注意时间复杂度
- 理解Java的集合库（HashMap、TreeMap、PriorityQueue）

### Python版
**优先级**：⭐⭐⭐
- 快速原型和思路验证
- 某些问题Python更简洁

**学习方式**：
- 复杂算法先用Python理清思路
- 再用Java实现优化版本

## 常见陷阱（避免）

❌ **不要**：刷题只看答案（要自己想）

✅ **要**：建立个人题库（记录思路和模板）
✅ **要**：定期复习（不要做完就忘）
✅ **要**：模拟真实面试环境
